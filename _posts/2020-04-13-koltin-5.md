## 확장함수

---

자바 VS 코틀린

기존에 만들어져 있는 클래스에 새로운 메서드를 추가 하려면,
- 자바 : 해당 클래스를 상속하는 새로운 클래스를 작성해야 한다.
- 코틀린 : **확장함수(extension function)**를 이용하여 상속 없이 기존 클래스에 새로운 함수를 추가 할 수 있다.

**확장함수의 정의**
```kotlin
fun String.toWeekDayWord(): String {
    val dayType = hashMapOf<String, String>(
        "MONDAY" to "월",
        "TUESDAY" to "화",
        "WEDNESDAY" to "수",
        "THURSDAY" to "목",
        "FRIDAY" to "금"
    )
    return dayType[this] ?: ""
}
```
* 리시버 타입(receiver type) : 확장 함수를 추가 할 대상 클래스
1. 리시버 타입 뒤에 점을 찍고 그 뒤에 원하는 함수의 형태를 찍는 방법으로 정의한다.
2. 함수 구현부에 this를 사용하여 클래스의 인스턴스에 접근하며, 이를 리시버 객체(receiver object)라고 한다.


**확장함수의 호출**

//예제1

fun String.toWeekDayWord(): String {
    val dayType = hashMapOf<String, String>(
        "MONDAY" to "월",
        "TUESDAY" to "화",
        "WEDNESDAY" to "수",
        "THURSDAY" to "목",
        "FRIDAY" to "금"
    )
    return dayType[this] ?: ""
}

//호출
var extTest = "MONDAY"

extTest = extTest.toWeekDayWord() //월

//예제2

fun String.tttest(str : String) = "$this$str"

//호출
var stttt = "gag"
stttt = stttt.tttest("호잉")

println(stttt) // gag호잉
