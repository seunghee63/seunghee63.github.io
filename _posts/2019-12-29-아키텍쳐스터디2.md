### Repository : 데이터가 로컬에서 오는지, 리모트에서 오는지 결정하고 데이터를 뿌려줌

#### 레포지토리 패턴

`Activity -> Repository -> Romote/Local DataSource `

* **장점**

1. Activity 의 코드 단순화
2.  코드 재사용
3. Repository 리모트, 로컬의 데이터에 대한 데이터를 구분해서  => Activity 의 코드 단순화22

* **Repository의 기능**

1. App 내부디비, 혹은 외부에서 가져오는지 판단/ 분기처리
2. 코드 재사용

* **기타**

1. RemoteDataSource/ LocalDataSource 는 데이처를 처리!
2. 실질적으로 데이터를 가지고 있는 것은 아님. 
3. 1) remote의 데이터를 2) Local에 저장해야하는 상황도 Repository가 처리함
